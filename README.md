# Cmail - 輕量級 Minecraft 郵件系統

Cmail 是一款專為 Paper 1.21.1 伺服器設計的郵件插件。它提供直覺的 GUI 介面、附件支援，並使用 SQLite 非同步儲存。

## 核心特色

* **直覺式 GUI 介面**：透過箱子介面管理郵件、查看收件箱及寄送附件。
* **附件支援**：寄信時可放入實體物品作為附件。
* **非同步 SQLite 儲存**：所有的資料庫讀寫操作均在非同步執行緒完成，防止伺服器卡頓。
* **自動化空間檢查**：領取附件前自動檢查玩家背包空間，並透過 GUI 物品 Lore 即時反饋。
* **登入提示**：玩家登入時自動提示未讀郵件數量。
* **高度配置化**：訊息與 GUI 標題皆可在 `config.yml` 中修改。

## 安裝要求

* **伺服器版本**：Paper/Spigot 1.21.1 或更高版本。
* **Java 版本**：Java 21。

## 指令說明

| 指令 | 說明 | 權限 |
| :--- | :--- | :--- |
| `/mail` | 開啟郵件管理主選單 (收件箱/寄信說明) | 無 (預設全玩家) |
| `/mail send <玩家ID> <訊息文字>` | 撰寫郵件並開啟附件放置介面 | 無 (預設全玩家) |

## 使用教學

### 寄送郵件
1. 輸入 `/mail send [玩家名稱] [文字]`。系統會自動驗證該玩家是否曾加入過伺服器。
2. 開啟 4 排空間的 GUI，前 3 排可放入附件物品。
3. 點擊第 4 排的 **[綠色玻璃]** 確認寄出，或 **[紅色玻璃]** 取消。

### 收取郵件
1. 輸入 `/mail` 並點擊 **[查看收件箱]**。
2. 在收件箱中點擊代表郵件的書本，附件將自動放入背包，郵件隨即銷毀。
3. 若背包空間不足，書本上會出現紅色警告提示。
